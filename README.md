# Plant-Pathology-2021---MultiLabel-Dataset
This is a repository of the training and test data used in the article, "Deep Learning for Apple Leaf Disease Diagnosis: A Comparative Study with CNNs and ViTs".

Note that this dataset is a conversion of a Multi-Class dataset to a Multi-Label dataset, and the train-test ratio is 80:20.

The original public dataset is collected by Thap et. al. as in: www.kaggle.com/competitions/plant-pathology-2021-fgvc8/overview/citation

It can be found here: https://kaggle.com/competitions/plant-pathology-2021-fgvc8
